# Yelp Bayesian Network Analysis

Probabilistic graphical model analysis of how business metadata correlates with customer ratings.

## Research Questions
* How do restaurant attributes (price, delivery) help predict whether a restaurant receives a high star rating?
* Given the a arbitrary volume of training data, how sensitive is a Bayesian network?

## Setup
```bash
pip install -r requirements.txt
```

## Dataset

- Yelp Open Dataset: https://business.yelp.com/data/resources/open-dataset/
